# 🤖 AI Resume Screener & Career Roadmap Advisor

> **PS-09 | CSE AI** - AI-powered resume evaluation and career guidance system for students

## 🎯 Problem Statement

Fresh graduates struggle to understand how their current skills compare to industry expectations. Recruiters spend hours manually screening resumes.

## 💡 Solution

An AI-powered system that evaluates resumes against industry benchmarks and generates personalized career roadmaps.

## ✨ Features

- 📄 Upload resume (PDF/DOCX/TXT)
- 🎯 Select from 8+ job roles
- 📊 Get instant resume score (0-100)
- 🔍 Identify skill gaps
- 💡 Receive personalized suggestions
- 🎓 Get certification recommendations
- 🗺️ Generate 18-month career roadmap
- 📥 Download analysis report

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **Backend:** Python
- **Libraries:** PyPDF2, python-docx, scikit-learn, pandas
- **Deployment:** Streamlit Cloud / Hugging Face
